## 문제

텍스트 기반 일기는 시각적으로 기록하기 어렵고 꾸준히 작성하기 지루할 수 있다.  
사용자가 짧은 일기만 입력해도 AI가 그림일기 형태로 자동 변환하여 더 재미있고 감성적으로 하루를 기록할 수 있도록 한다.  
AI 이미지 생성을 활용하여 사용자의 경험을 시각적으로 표현하는 것이 목표이다.

---

## 사용자

감성적인 기록을 좋아하는 일반 사용자 및 그림일기를 간단하게 만들고 싶은 사용자

---

## 핵심 기능 (3개 이하)

1. 기능 1 - 사용자가 하루 일기를 텍스트로 입력
2. 기능 2 - LLM이 그림 생성용 프롬프트 자동 생성
3. 기능 3 - AI가 그림일기 스타일 이미지 출력

---

## 입력 → AI 처리 → 출력

사용자 일기 텍스트 입력 → OpenAI API가 그림 설명 프롬프트 생성 → HuggingFace 이미지 생성 모델이 그림일기 이미지 출력

---

## 기술 스택

- **UI**: Next.js 16 + React 19 + Tailwind CSS v4 (커스텀 글래스모피즘·셰이더 디자인)
- **모델 / API**: Google Gemini 2.5 Flash (한국어 일기 → 이미지 프롬프트 변환), Pollinations.ai Flux 모델 (그림일기 이미지 생성)
- **외부 서비스**: HuggingFace Spaces, GitHub Actions

---

## 데이터 출처

사용자가 직접 입력한 한국어 일기 텍스트. 
이미지는 Pollinations.ai API를 통해 실시간 생성되며, 별도 데이터셋 없음. 
데이터는 사용자 브라우저 localStorage에만 저장되며 서버에 보관되지 않음.

---

## 완료 조건

사용자가 한국어 일기 텍스트를 입력하면 30초 이내에 Gemini가 이미지 프롬프트를 생성하고, Pollinations Flux 모델이 그림일기 스타일 이미지를 출력한다.
